# Sudhir Singh (Fauji) Website

Static bilingual campaign website for Sudhir Singh (Fauji), Pradhan Pad Pratyasi, Gram Kapoorpur, Tehsil Meerganj, Bareilly.

## Files

- `index.html` - website structure
- `styles.css` - responsive design
- `script.js` - Hindi/English switch and WhatsApp contact form
- `assets/` - portrait and social preview images
- `privacy-policy.html`, `ads.txt`, `robots.txt`, `sitemap.xml` - AdSense/search readiness files

## GitHub Pages

Publish from the repository's `main` branch, root folder.

For AdSense, replace the placeholder line in `ads.txt` with the publisher line Google gives you after you create an AdSense account/property.
